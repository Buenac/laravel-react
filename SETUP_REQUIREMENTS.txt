INSTALLATION STEPS
------------------
1. Clone repository:
   git clone https://github.com/Buenac/laravel-react.git
   cd laravel-react

2. Install PHP dependencies:
   composer install

3. Copy environment file:
   Windows: copy .env.example .env
   Linux/Mac: cp .env.example .env

4. Generate application key:
   php artisan key:generate

5. Configure database in .env:
   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=laravel_weather
   DB_USERNAME=root
   DB_PASSWORD=your_password

6. Create database:
   CREATE DATABASE laravel_weather;

7. Run migrations:
   php artisan migrate

8. Add OpenWeather API key to .env:
   OPENWEATHER_API_KEY=your_key_here

9. Install frontend dependencies:
   npm install --legacy-peer-deps

10. Start Laravel server (Terminal 1):
    php artisan serve --port=8000

11. Start Vite dev server (Terminal 2):
    npm run dev

12. Access application:
    http://127.0.0.1:8000

- DB_HOST=127.0.0.1
- DB_PORT=3306
- DB_DATABASE=laravel_weather
- DB_USERNAME=root

- OPENWEATHER_API_KEY=your_api_key_here


DATABASE SCHEMA
---------------
Tables:
1. users
   - id, name, email, password, email_verified_at, remember_token, created_at, updated_at

2. comments
   - id, text, created_at, updated_at

3. weather_favorites
   - id, user_id (FK), date, note, created_at, updated_at

4. personal_access_tokens (Sanctum)
   - id, tokenable_type, tokenable_id, name, token, abilities, last_used_at, expires_at, created_at, updated_at


API INTEGRATIONS
----------------
OpenWeatherMap API:
- Endpoint: https://api.openweathermap.org/data/2.5/forecast/daily
- Parameters: q (city), cnt (days), appid (key), units (metric)
- Cache Duration: 3600 seconds (1 hour)
- Free API Key Registration: https://openweathermap.org/api

TROUBLESHOOTING
---------------
Issue: Composer install fails
Fix: Check PHP version (php -v), enable required extensions in php.ini

Issue: npm install fails
Fix: Use --legacy-peer-deps flag: npm install --legacy-peer-deps

Issue: Vite manifest not found
Fix: Ensure npm run dev is running or run npm run build

Issue: Weather API error
Fix: Check OPENWEATHER_API_KEY in .env, run: php artisan cache:clear

Issue: Database connection error
Fix: Verify DB credentials in .env, ensure MySQL is running

Issue: Port already in use
Fix: Change port: php artisan serve --port=8001
